ascending.js

¶

This is a terse function that provides a partial ordering for a poset[1]. It uses the javascript conditional operator several times in a nested fashion to determine the order of the two given elements. 

             		           
                               
                            
                 ¶               
               
The ternary expressions is equivalent to the following code: 



         if (a < b){
           return -1
         }
         else{
           if (a > b){
             return 1
           }
           else{
             if ( a>= b)
               return 0
             else
               return NaN
           }
         }

             		                            
d3.ascending = function(a, b) {
  return a < b ? -1 : a > b ? 1 : a >= b ? 0 : NaN;
};
